# Async-Script-Loader
A Promise based script loader with in built caching

## To Install
``` 
npm i async-script-loader
```

## To Use
``` 
import scriptLoader from '@script-loader'

scriptLoader([], options = {})

```

## Options value
``` options : {
  "async": boolean,
  "defer": boolean,
}
```
